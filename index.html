⌨ SLASH COMMAND CHEAT SHEET

/login

Authenticate with your GitHub account

/model

Switch the active AI model mid-session

/logout

Sign out of the current session

/mcp

Add, remove, or configure MCP servers

/session

View what commands are currently allowed

/feedback

Send feedback directly from the CLI

/reset

Reset all permissions for a fresh start

/help

See available commands and usage hints

/add-directory

Allow Copilot to read a new directory

Ctrl+R

Review logs of commands run on your behalf

💡 Pro tip: Type / at any prompt to see the full command menu. You can @-mention files to focus Copilot on specific code.
